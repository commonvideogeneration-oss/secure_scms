# COIT20246 – Networking and Cyber Security Project

## Project Overview
This repository contains the group project for the unit **COIT20246 – Networking and Cyber Security**. The project focuses on the design of a secure, scalable network and cloud-based infrastructure for **Truelec**, an Australian electrical contracting company that relies on a centralised booking system to coordinate operations across multiple locations.

The design addresses enterprise networking requirements, cloud service evaluation, cybersecurity risk management, ethical considerations, and professional reflection.
